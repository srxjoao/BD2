## 🗂 Executar Projeto

```bash
# Clonar Projeto.
$ git clone https://github.com/srxjoao/BD2.git

# Entrar na pasta server.
$ cd server

# Baixar as dependencias.
$ npm install

# Executar.
$ node server.js

# Novo cmd.
# Entrar na pasta client.
$ cd client

# Baixar as dependencias.
$ npm install

# Executar.
$ npm run dev

 ```
